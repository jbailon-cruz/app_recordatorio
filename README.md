# App de Recordatorios (Flutter)

Aplicación Flutter básica que permite activar/desactivar un recordatorio, mostrar un SnackBar y navegar a una segunda pantalla con detalle del recordatorio.

## Funcionalidades
- Cambio de título, AppBar y texto principal
- SwitchListTile para activar/desactivar recordatorio (widget nuevo)
- Botón funcional con SnackBar
- Navegación a segunda pantalla y botón para regresar
